⏰ Deadlines
Pre-tour Picks must be entered before 10:00 on 4 July. Fields lock at the deadline — any empty field at that point is recorded as no selection.
Daily stage Picks must be entered before 10:00 on each stage day. Fields lock at the deadline. Empty fields at 10:00 are recorded as DNS (0 points). Late input is not accepted.
If 1 or 2 players are tied for the lead going into Stage 21, those players must submit by 09:45 on 26 July.
🃏 The Joker
Each player has one Joker for the entire tour. Tap the 🃏 button next to a stage input to activate it — it glows purple when active. Tap again to deactivate.
If your Joker pick is correct → 2 points instead of 1.
If you DNS the stage where your Joker is set → Joker is cancelled and lost.
Players may select the same riders as each other — no restrictions.
🛡️ Spoiler Shield
The 🛡️ Spoiler Shield button sits flush against the 👁️ All Picks tab in the navigation bar, so it is clear they are associated. Tap it to activate — it turns red when on.
When active — stage results and picks in the All Picks table are hidden from the moment of activation onwards. Past results (stages already revealed before activation) remain fully visible.
When deactivated — all results become visible simultaneously, including any stages that accumulated while the shield was active.
The Spoiler Shield only hides stage results in All Picks. It does not affect the Leaderboard, Points Breakdown, or Live Race Standings — simply avoid clicking those tabs if you do not want to see them.
The shield persists across page loads and browser restarts — it remains active until you explicitly tap the button to deactivate it. Refreshing the page will not turn it off.
📢 Media Centre
The 📢 Media tab is a shared message board for all three players on this device. Use it for admin announcements, game updates, feature requests, technical notices, and player banter.
Select your name from the dropdown, type your message, and tap Post. Messages are stored locally and visible to whoever opens the app on this device.
Admin posts are used to notify players of changes such as new Last KM images being added, scoring corrections, or rule clarifications.
🔒 Privacy & pick visibility
✏️A player can change their stage pick any number of times before the deadline. The most recent entry is what counts. Once the deadline passes the field locks and no further changes are possible.
🔒Before the deadline — a player's stage pick is visible only to themselves. If another player views that player's stages, they see "Pick hidden until the deadline".
🔓After the deadline — the pick is revealed to all three players automatically in the Stages tab and the All Picks table.
🎽Pre-tour picks — all pre-tour selections remain hidden from other players until 10:00 on 4 July, at which point all selections are revealed simultaneously.
🚫No editing other players' picks — the app only allows a player to edit their own picks. Attempts to enter a pick under another player's name are blocked.
🏮DNS (Did Not Sign-in) — if a stage pick field is empty when the deadline passes, a DNS is recorded automatically. A DNS scores 0 points. If a Joker was active on that stage it is cancelled and cannot be used again.
